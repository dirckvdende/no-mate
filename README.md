
# No Mate: A Chess-Like Puzzle Game

This is the source code for a small puzzle game called "No Mate". The goal of
the puzzles is to place chess pieces on a playing field such that none can
attack each other.

## Contributing Puzzles

Would you like to contribute a puzzle? Please look at the format the puzzles are
stored in in [src/puzzles](./src/puzzles), as well as the README in that
folder. Create a pull request to suggest the addition of one or more puzzles.
