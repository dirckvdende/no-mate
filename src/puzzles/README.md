
# Directory of Puzzles

This folder contains all puzzles. These are stored in files with names [files/1.ts](./files/1.ts), [files/2.ts](./files/2.ts), etc. To add a puzzle, simply create a file with a name that is the next available number.

New/custom puzzle pieces can be added in [pieces.ts](./pieces.ts), although specific features may require changes in other parts of the codebase.
