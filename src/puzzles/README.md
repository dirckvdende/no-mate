
# Directory of Puzzles

This folder contains all puzzles. These are stored in files with names `1.ts`, `2.ts`, etc. To add a puzzle, create a file with a name that is the next available number, and add the puzzle import to [puzzles.ts](./puzzles.ts).

New/custom puzzle pieces can be added in [pieces.ts](./pieces.ts), although specific features may require changes in other parts of the codebase.
